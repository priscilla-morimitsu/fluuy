# Conversion UX Optimizer

Improve conversion without sacrificing trust, clarity or accessibility.

## Use for

- landing pages
- hero sections
- pricing pages
- signup flows
- checkout
- trial activation
- upgrade flows
- lead forms
- waitlists
- product pages

## Principles

- Clarity before persuasion
- One primary CTA per section
- Reduce friction
- Address objections
- Show concrete value
- Use social proof responsibly
- Avoid dark patterns
- Keep forms short
- Make pricing transparent
- Explain what happens next

## Review checklist

Check:

- Is the value proposition clear?
- Is the CTA specific?
- Is there unnecessary friction?
- Are objections answered?
- Is the layout scannable?
- Is trust visible?
- Is accessibility preserved?
- Is the visual style aligned with the design system?
- Is Liquid Glass used subtly and without glow?

## Output format

Deliver:

1. conversion diagnosis
2. friction points
3. proposed improvements
4. rewritten copy when needed
5. UI structure recommendations
6. ethical risks to avoid
